Play famous "Descer Pra BC" sound.

https://www.youtube.com/watch?v=r6kXxV-yPDA


## Compatibility

- Linux
- Windows
- Mac

## Installing

Install globally:

    npm install -g desceprabc

## Running

With npm:

    npm install -g desceprabc
    desceprabc

Npx:

    npx desceprabc

Node script:

    import desceprabc from 'desceprabc';

    desceprabc();
